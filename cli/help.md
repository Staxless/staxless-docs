# staxless help

> Get help on any command or topic.

---

## Usage

```bash
staxless help [topic]
```

---

## Examples

### General help

```bash
staxless help
```

### Help for specific command

```bash
staxless help dev
staxless help generate
staxless help init
```

### Command-specific help

Also works with `--help`:

```bash
staxless dev --help
staxless generate service --help
```

| Topic | Description |
